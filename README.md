# Movie-Card

- React 연습용 개인 프로젝트
- 영화 제목, 별점, 리뷰를 입력하여 카드를 생성하고 이미지로 저장/공유할 수 있다.
- 지속적인 기능 및 사용자 친화 디자인 추가 : [개발 일정 관리](https://kanghyeyoon.notion.site/31bfa82984a04cb68bf31908ceb60624)
